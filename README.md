# ECS 世界模拟系统 v4.16.0
自洽运转的沙盒演化世界，参考WorldBox的完全自发演化体验，零硬编码干涉，所有规则自底向上涌现。

## ✨ 核心特色
- 🧬 **完全自发演化**：无人工预设食物链、物种、捕食关系，生态与文明完全由自然选择和主观能动性自发生成
- 🚫 **零硬编码规则**：没有预设技术树、文明阶段、社会制度，所有上层建筑由人类个体实践自然产生
- 🔄 **全链路物质循环**：从水循环、土壤养分、植物生长、动物捕食到尸体分解，完全闭环无人工兜底补给
- 📚 **可失传技术体系**：技术由人类个体主动发明，通过人际传授、代际传承、书籍记录传播，掌握者全部死亡则技术永久失传
- 🌡️ **完整物理环境**：包含水循环、温度传导、风场、土壤侵蚀、养分循环、极端气候等真实物理系统
- 🎮 **可视化Demo**：内置像素风实时可视化界面，支持暂停、调速、步进、纪元重置等控制功能
- ⚡ **高性能架构**：v4版本架构支持100x100网格下1000+实体稳定运行，单步耗时<50ms

## 🚀 快速启动
### Windows 一键启动
直接双击项目根目录下的 `start_ecs_demo.bat`，脚本自动清理残留进程、启动前后端服务，完成后访问：
```
http://localhost:3000
```
一键停止服务：双击 `stop_ecs_demo.bat` 即可自动关闭所有相关进程。

### 手动启动
```bash
# 1. 启动后端服务
cd presentation/visualization
python demo_server.py

# 2. 启动前端
cd ../../frontend
npm run dev
```
访问 http://localhost:3000 即可进入Demo界面。

## 📦 核心模块
### 1. 核心架构
- **ECS框架**：纯组件+系统模式，数据与逻辑完全分离，支持零代码扩展组件与系统
- **实体池**：高性能实体复用机制，避免频繁创建销毁开销
- **查询缓存**：精确失效的查询缓存，大幅提升组件查询性能
- **系统调度器**：支持按优先级、间隔调度系统，支持按类型/名称查询系统

### 2. 环境物理系统
- **水循环**：降雨→下渗→蒸发→径流→地下水完整循环，对接雨水、土壤、水坑系统
- **温度系统**：太阳辐射加热、层间热传导、昼夜温差、季节变化，支持5层土壤温度存储
- **风场系统**：动态风场生成、热量平流、种子/水汽/污染物扩散
- **土壤系统**：通用土壤流失方程模拟侵蚀，氮/磷/碳养分循环，微生物分解机制
- **极端气候**：干旱/洪水/寒潮/热浪/野火/冰雹/龙卷风/飓风共8种极端事件，基于环境条件自发生成

### 3. 生态系统
- **植物生长**：完全依赖土壤湿度、养分、温度、光照，符合最小因子定律，成熟后自动结果
- **动物行为**：食草动物啃食植物、食肉/杂食动物捕食猎物，成功率由体型、速度、食性偏好动态决定
- **尸体分解**：多类尸体分5阶段分解，释放养分到土壤，有毒尸体传播疾病，打通生物→死亡→土壤→植物物质循环
- **微生物系统**：分解者、固氮菌、致病菌、共生菌、菌根真菌，加速有机质分解、补充土壤肥力
- **可演化性状**：动物核心性状（食性偏好、体型、速度、繁殖率、变异率）全部可遗传、可变异，性状差异超50%自动生成新物种

### 4. 文明系统
- **人口全生命周期**：出生→成长→工作→社交→生育→衰老→死亡完整闭环
- **需求驱动行为**：人类行为完全由饥饿、口渴、温度、社交、繁衍、安全等基础需求驱动
- **动态文明阶段**：6大维度18个指标加权评分自动匹配文明阶段，无硬性单一指标，支持文明降级
- **文化演化**：7大价值观维度，支持规范/传统/禁忌动态管理，跨文明文化传播
- **文明特质**：20种预设特质（生产/社会/生存/军事/特殊5大类），自动属性修正

### 5. 技术体系（完全无硬编码）
- **个体创新**：技术由人类个体基于实践、需求、创造力主动掌握，系统仅做创新成功率判断
- **技艺传授**：近距离社交触发教学，亲属/代际传承+40%学习概率、同部落+20%学习概率，幼年学习速度+50%
- **书籍传承**：大师级技能拥有者可将技能写成书籍，其他人类可通过阅读书籍学习技能，实现技术脱离人类实体传承
- **技术失传**：掌握某技术的最后一人死亡则技术从文明库永久消失，无人工兜底恢复机制

## 🎯 核心设计原则
1. **纯ECS架构**：组件仅存数据，所有业务逻辑放在System中，组合优于继承，注册表模式零代码扩展
2. **零硬编码规则**：没有预设的"正确"演化路径，所有上层结构完全自下而上涌现
3. **无人工兜底**：允许资源匮乏、种群灭绝、文明消亡，所有结果完全符合自然逻辑
4. **渐进式兼容**：旧组件提供兼容层，不强制修改所有业务系统，支持平滑迭代
5. **性能优先**：所有优化保持API完全兼容，在不影响体验的前提下最大化运行效率

## 📊 性能指标
- 支持网格规模：最大1000x1000
- 支持实体数量：10000+（100x100网格下）
- 单步平均耗时：<30ms（100x100网格，50人类+200动植物实体）
- 峰值内存占用：<80MB（2000步模拟）
- 长时稳定性：2000步连续运行无崩溃，资源闭环无泄漏
